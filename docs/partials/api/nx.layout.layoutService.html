layoutService
service in module nx.layout

Description

Service that can be used to set document layout. This layout is set in $rootScope variables and can be set as a class on the document and used to make layout styling or inclusion changes.

See the module description for more information.

Methods

  • setLayout(layout)

    Sets a custom layout. To set the default layout, pass 'default' as the layout name.

    Parameters
    ParamTypeDetails
    layoutstring

    The name of the layout to set
